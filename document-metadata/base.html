base

Specifies the address (URL) used by all relative addresses contained within a document. There can only be one <base> element in the document. If there is more than one, only the first is considered.

href

The base URL to be used throughout the document for relative URL addresses. If specified, this element must come before any others with attributes whose values are URLs.

target

Default location to display the result when hyperlinks are triggered, by elements that do not have an explicit target reference.

_self

Puts the result in the same tab. Standard value.

_blank

Puts the result in a new tab.

_parent

Places the result in the parent tab set. If there is no parent, it behaves like the _self form.

_top

Places the result in the top-level navigation context, that is, it is an ancestor of the current one, not the parent. If there is no parent, this option behaves the same as _self.
